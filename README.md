# portal-cns-resource

## 项目简介

portal-cns-resource 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定的域名或网站，仅作为静态资源文件的集中存储与分发渠道。

所有页面均为独立、自包含的 HTML 文件，适用于快速展示、存档或临时访问场景。

## 目录结构

```
portal-cns-resource/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 共享资源文件（如 CSS、JS、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本说明文档
```

- `pages/`：每个子目录或文件对应一个独立页面，命名方式为 `page-<名称>.html`。
- `assets/`：存放页面间共享的静态资源，按类型或用途组织。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，包含所需样式与脚本（或引用 `assets/` 下资源）。
- 页面内容不涉及任何具体的商业、营销或特定网站信息。
- 归档目的：便于版本管理、历史回溯与多场景分发。

## 使用方式

- 可通过 GitHub Pages 或其他静态托管服务直接部署本仓库。
- 也可将 `pages/` 下的文件单独下载至本地打开使用。
- 如需引用共享资源，请确保路径指向 `assets/` 目录。

## 维护说明

- 本仓库由维护者按需更新，不承诺固定更新频率。
- 欢迎提交 Issue 或 Pull Request，但请注意：新页面应保持独立、自包含，不引入外部依赖或动态加载。
- 提交前请确保文件命名清晰，避免与已有页面冲突。
